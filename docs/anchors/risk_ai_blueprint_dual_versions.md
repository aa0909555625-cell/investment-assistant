# 投資 AI 風控工具｜整體藍圖封存
## 【雙版本結構】內部專業版 × 對外展示版

本文件為系統級定錨藍圖，用於後續所有設計、實作、討論與擴充。  
明確區分「對外展示」與「內部專業」，兩者永久不混用。

---

## 一、整體系統共用核心（兩版共用，不可分叉）
**Core｜風控與資料基礎（不可對外暴露）**

### 市場資料蒐集層
- 價格
- 成交量
- 波動指標
- 基本市場行為資料

### 風控引擎（AI / Rule-based 混合）
- 風險分數計算
- 波動異常偵測
- 禁止交易判斷
- 風控門檻設定（不可即時改）
  - 波動上限
  - 最大回撤容忍
  - 連續虧損保護

### 風控事件日誌（ELOG）
- 當下數值
- 判斷原因
- 後續市場結果
- 版本控管
  - 規則版本
  - 風控版本
  - 回測對應版本

> ⚠️ 此層 任何版本都不能被一般使用者操作或修改

---

## 二、】【對外展示版】封存清單
**定位：風險觀測 × 教育展示 × 防誤解**  
**對象：親友（如阿姨）、非專業者、潛在合作對象**  
**核心原則：不預測、不承諾、不給操作權**

### A. 功能層（只讀）
- 市場風險狀態顯示：低風險 / 中風險 / 高風險
- 市場穩定度描述：穩定 / 波動 / 異常
- 風險變化趨勢（歷史回顧）：過去風險升高 → 實際市場狀況

### B. 呈現層（UI / UX）
- 顏色：中性、低刺激
- 無即時跳動、無紅綠燈交易暗示
- 不顯示任何：進場 / 出場 / 買賣 / 勝率 / 報酬率

### C. 說明層（固定文案）
- 系統定位說明：「本系統為風險觀測工具」
- 非投資建議聲明（白話版）
- 生活化比喻：氣象預報 / 交通警示
- 明確說明：
  - 不能用來決定買賣
  - 不能保證任何結果

### D. 權限層
- 使用者僅能觀看
- 無任何參數調整權
- 無任何策略選擇權

### E. 封存狀態標記
- 【對外展示版】
- 只讀 / 不可操作
- 不構成投資建議

---

## 三、】【內部專業版】封存清單
**定位：決策依據 × 交易輔助 × 團隊可用系統**  
**對象：你本人、未來內部成員**  
**核心原則：規則至上、可回測、可交接**

### A. 風控決策層（核心）
- 量化風控指標：波動值 / 回撤值 / 風險分數
- 交易狀態判斷：可交易 / 降倉 / 禁止交易
- 連續虧損保護機制
- 極端事件處理邏輯

### B. 進出場與資金管理（內部專用）
- 進場條件（系統判斷）
- 停損規則（不可臨時改）
- 停利規則
- 倉位上限
- 單日 / 單週風險上限

### C. 回測與驗證層
- 策略回測
- 風控回測
- 實盤 vs 回測對照
- 極端行情檢驗

### D. 操作與權限設計
- 角色區分：風控設計者 / 系統維護者 / 操作執行者
- 規則變更流程：提出 → 回測 → 記錄 → 上線
- 禁止即時熱修規則

### E. 紀錄與成長資產
- 交易紀錄
- 風控事件紀錄
- 決策檢討紀錄
- 版本演進歷史

### F. 封存狀態標記
- 【內部專業版】
- 可操作 / 有權限控管
- 僅供內部決策與研究

- 對外版 永不顯示：進出場點 / 勝率 / 報酬 / 策略
